Binary Tree

A Binary Tree is a hierarchical data structure where each node has at most two children, referred to as the left child and right child.

Key Properties​

  • The topmost node is called the root.
  • Nodes with no children are called leaf nodes.
  • The depth of a node is the number of edges from the root to that node.
  • The height of a tree is the number of edges in the longest path from root to a leaf.

Structure of a Binary Tree​

Types of Binary Trees​

  1. Full Binary Tree – Every node has either 0 or 2 children.
  2. Complete Binary Tree – All levels are completely filled except possibly the last, which is filled from left to right.
  3. Perfect Binary Tree – All internal nodes have two children, and all leaves are at the same level.
  4. Degenerate (Skewed) Binary Tree – Each parent has only one child (like a linked list).

Catalan Number and Binary Trees​

The Catalan number counts the number of possible distinct Binary Search Trees (BSTs) that can be formed with n distinct keys.

Formula:

Cn=(2n)!(n+1)!⋅n!C_n = \frac{(2n)!}{(n+1)! \cdot n!}Cn​=(n+1)!⋅n!(2n)!​

Or recursively:

Cn=∑i=0n−1Ci⋅Cn−1−iC_n = \sum_{i=0}^{n-1} C_i \cdot C_{n-1-i}Cn​=i=0∑n−1​Ci​⋅Cn−1−i​

where C0=1C_0 = 1C0​=1.

Example: For n=3n = 3n=3 keys,

C3=6!4!⋅3!=72024⋅6=5C_3 = \frac{6!}{4! \cdot 3!} = \frac{720}{24 \cdot 6} = 5C3​=4!⋅3!6!​=24⋅6720​=5

So, 5 distinct BSTs can be formed.

Number of Labelled Binary Trees​

If we have n labelled nodes (distinct labels), the number of labelled binary trees is:

Count=Cn⋅n!\text{Count} = C_n \cdot n!Count=Cn​⋅n!

Where:

  • CnC_nCn​ = Catalan number (structure count)
  • n!n!n! = permutations of labels

Example: For n=3n = 3n=3,

C3=5,n!=6C_3 = 5, \quad n! = 6C3​=5,n!=6

Total labelled binary trees:

5⋅6=305 \cdot 6 = 305⋅6=30

Max and Min Number of Nodes Given Height​

Let height hhh be number of edges in longest path from root to leaf.

  • Max Nodes in height hhh (Perfect Binary Tree):
Nmax=2h+1−1N_{\text{max}} = 2^{h+1} - 1Nmax​=2h+1−1
  • Min Nodes in height hhh (Skewed Tree):
h+1h + 1h+1

Example: For h=3h = 3h=3, Max nodes:

23+1−1=152^{3+1} - 1 = 1523+1−1=15

Min nodes:

3+1=43 + 1 = 43+1=4

Max and Min Height Given Number of Nodes​

Let nnn = total nodes.

  • Min Height (Perfect Binary Tree):
hmin=⌊log⁡2(n)⌋h_{\text{min}} = \lfloor \log_2(n) \rfloorhmin​=⌊log2​(n)⌋
  • Max Height (Skewed Tree):
hmax=n−1h_{\text{max}} = n - 1hmax​=n−1

Example: For n=10n = 10n=10, Min height:

⌊log⁡2(10)⌋=3\lfloor \log_2(10) \rfloor = 3⌊log2​(10)⌋=3

Max height:

10−1=910 - 1 = 910−1=9

Internal and External Nodes​

Definitions​

  • Internal Node: A node with at least one child (non-leaf).
  • External Node: A node with no children (leaf).

Properties in a full binary tree:

E=I+1E = I + 1E=I+1

where:

  • EEE = number of external nodes
  • III = number of internal nodes
